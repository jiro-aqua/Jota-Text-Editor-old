Jota Text Editor

Jotaは、長文テキストの編集向けに設計されたテキストエディタです。
「イオタ」と読んでください。

★現在αテスト中です。以下の条件で構わない人だけダウンロードしてください。
・落ちても泣かない
・バグっぽい挙動を見つけたら、作者に連絡する
・欲しい機能があったら、作者に連絡する


機能

・100万文字程度までのテキストファイルに対応(Alphabetで1MBytes/ShiftJISで2MBytes)

・文字コード自動判定（以下の文字コードを自動判定）
  UTF-8
  UTF-16BE
  UTF-16LE
  UTF-32BE
  UTF-32LE
  WINDOWS-1251
  WINDOWS-1252
  WINDOWS-1253
  WINDOWS-1255
  SHIFT_JIS
  ISO-2022-JP
  ISO-2022-CN
  ISO-2022-KR
  ISO-8859-5
  ISO-8859-7
  ISO-8859-8
  BIG5
  GB18030
  EUC-JP
  EUC-KR
  EUC-TW
  IBM855
  IBM866
  KOI8-R
  MACCYRILLIC
  HZ-GB-2312
  X-ISO-10646-UCS-4-3412
  X-ISO-10646-UCS-4-2143
  (実際にデコード可能かどうかは、Androidの実装に依存します）

・文字コードの変更
・改行コード自動判定、保存、変更
・カーソル位置の保存
・ファイル履歴
・ファイル内検索(正規表現対応)

・外部アプリ連携
  マッシュルーム
  テキストの共有
  外部アプリによる検索
  外部アプリで開く

  ★任意のアプリを直接呼び出すダイレクトインテント機能を搭載

・サムをドラッグすることによるファイル内移動
・フリックによる高速スクロール
・アンドゥ
・左右Alt/Ctrlキー同時押しによるショートカットキー(キー毎に割り当て有無を設定可能)
・色のカスタマイズ対応
・テキストファイルのショートカットアイコン作成機能
・ファイルエクスプローラ系アプリ、コンテンツプロバイダ系アプリからの呼出に対応
・テキスト選択用のマーカー機能搭載
（テキストを選択したエリアをタップすることで表示されます）


・対応OS Android 1.6/2.1/2.2/2.3
  (IS01,HTC Desire,005SH,Nexus One で動作確認）

・今後も含め、有料化および広告付与は行いません。
・安定したらオープンソースとして配布する予定です。

Jota Text Editor

Jota is a text editor designed for a long text file.
It's pronouced as "i-o-ta".

★Jota is currently in Alpha-test phase. 
Only in the case you can agreed below statement, you can use this.
- Don't cry when Jota crashed and lost your text.
- Mail me when you found a bug.
- Mail me new feature you want.

Functions
・Support until 1 million characters.

・Auto-Detect multiple character codes.
  UTF-8
  UTF-16BE
  UTF-16LE
  UTF-32BE
  UTF-32LE
  WINDOWS-1251
  WINDOWS-1252
  WINDOWS-1253
  WINDOWS-1255
  SHIFT_JIS
  ISO-2022-JP
  ISO-2022-CN
  ISO-2022-KR
  ISO-8859-5
  ISO-8859-7
  ISO-8859-8
  BIG5
  GB18030
  EUC-JP
  EUC-KR
  EUC-TW
  IBM855
  IBM866
  KOI8-R
  MACCYRILLIC
  HZ-GB-2312
  X-ISO-10646-UCS-4-3412
  X-ISO-10646-UCS-4-2143
  (Whether you can decode or not depends on Android's implementation.)

・Change character codes.
・Auto-Detect linebreak code.
・Change linebreak code.
・Keep linebreak code until saving a file.
・Save position of the cursor with file.
・Save history of recened files.

・Search a word in a file. (Support Regular Expression)

・Text sharing with external application.
  Mushroom
  Share (ACTION_SEND)
  Search (ACTION_SEARCH)
  Open (ACTION_VIEW)

  ★Direct Intent : You can launch any one application on menu item.

・Scroll with Thumb-drag.
・Scroll with Flick.
・Undo
・Support a Shortcut key with ALT/CTRL Key.
・Customize colors.
・Create a shortcut icon on HOME app.
・Customizable shortcut key bind.
・Text selector like Gingerbread.
(Double tap to select the word , then touch selected area to show marker.)

・Support Android OS 1.6 or later.
・Support both touch-screen device and qwerty-keyboard device.

・Forever FREE and NO-AD.
・In the future , source code will be opend unde Apache License.


CAUTION
  RTL-Language and bi-directional text isn't be supported.


★If you need a reply from me , DON'T put it in Market's Review .
Instead please let me know by E-mail or Twiter(@jiro_aqua)
You can send a message to me from Jota's Preferences Screen.

★In landscape orientation, you can't edit correctly because software keyboard will cover all of the screen.
This is a issue of the software keyboard.
I recommend to use 3rd party's Keyboard app like Grafitti.

★If you need grep (search from multiple text files), try my app "aGrep".

★If you couldn't load text file, please contact me!
